# Sudoku Solver

Simple C Program to solve sudoku puzzles. The program takes a text file as input and outputs the solved puzzle to the console. 

Compile the program:
```shell
gcc sudoku.c -o sudoku 
```
Run the program:
```shell
./sudoku.exe
```
